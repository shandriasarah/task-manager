# 📝 Gerenciador de Tarefas

Aplicação web para organizar tarefas do dia a dia, desenvolvida em React como projeto prático para consolidar conceitos fundamentais da biblioteca.

🔗 **Acesse a demo:** https://curso-de-react-dusky.vercel.app

## ✨ Funcionalidades

- ➕ Adicionar tarefas com título e descrição
- ✅ Marcar tarefas como concluídas
- 🗑️ Remover tarefas da lista
- 💾 Lista persistida na tela em tempo real

## 🛠 Tecnologias

- **React** — biblioteca para construção de interfaces
- **Vite** — build tool moderna e rápida
- **JavaScript (ES6+)**
- **CSS** — estilização customizada
- **Vercel** — hospedagem e deploy contínuo

## 💻 Como rodar localmente

```bash
# clone o repositório
git clone https://github.com/shandriasarah/curso-de-react.git

# entre na pasta
cd curso-de-react

# instale as dependências
npm install

# rode o projeto
npm run dev
```
